# 睡眠与情绪调节

睡眠和情绪会相互影响。压力、焦虑、反复思考和白天缺少恢复会让人更难入睡；睡眠不足又会降低情绪调节能力，使人更容易烦躁、低落、敏感或注意力下降。

常见睡眠困扰包括：入睡困难、半夜醒来、醒得太早、睡醒后仍然疲惫、睡前大脑停不下来。陪伴式回应中，应避免简单说“别想了”或“早点睡”，因为这可能让用户更有压力。更合适的方式是承认睡不着很消耗，并提供温和的睡前调整方法。

睡前稳定建议：

1. 固定睡前过渡  
睡前 30 分钟开始降低刺激，例如调暗灯光、减少工作学习信息、离开争论性聊天或高强度短视频。

2. 写下未完成事项  
如果脑中一直想着明天的任务，可以在纸上写下“明天第一步做什么”。这会给大脑一个结束信号。

3. 放松身体  
可以进行缓慢呼吸、轻柔伸展或渐进式肌肉放松。目标不是强迫自己立刻睡着，而是让身体从紧张转向休息。

4. 减少睡前自责  
越担心“我必须马上睡着”，越可能保持清醒。可以把目标从“必须睡着”改成“让身体休息一会儿”。

5. 保持白天节律  
尽量保持固定起床时间，白天接触自然光，适度活动。午睡不宜过长，避免影响夜间睡眠。

当用户说“我经常睡不着”时，可以回应：听起来你已经被这种疲惫拖了一段时间；睡不着不是你不努力，很多时候是压力和身体警觉状态没有降下来；今晚可以先尝试一个低要求的方法，比如写下明天最小的一步，再做 5 轮慢呼吸。

如果长期失眠持续数周，或伴随明显情绪低落、强烈焦虑、功能受损，建议用户寻求专业帮助。
