# 人际沟通与边界表达

人际困扰常见于朋友、伴侣、同学、同事和家庭关系中。很多冲突并不只是“谁对谁错”，还涉及需求没有被听见、边界不清楚、表达方式过于防御或长期压抑后的爆发。

陪伴式回应人际问题时，可以先帮助用户区分事实、感受、需求和请求。事实是发生了什么；感受是自己内在的情绪；需求是自己重视什么；请求是希望对方做什么。

表达模板：

1. 描述事实  
“当你连续几次临时取消约定时……”

2. 表达感受  
“我会感到失落，也有点不被重视。”

3. 说明需求  
“我需要关系里有一些稳定和提前沟通。”

4. 提出请求  
“下次如果你来不了，可以提前告诉我吗？”

边界表达不是攻击别人，而是清楚说明自己能接受什么、不能接受什么。好的边界通常具体、平静、可执行。例如：“我愿意听你说这件事，但如果开始人身攻击，我会先暂停这次聊天。”

常见沟通建议：

1. 避免使用“你总是”“你从来不”这类绝对化表达。
2. 尽量用“我感到……”开头，而不是直接给对方贴标签。
3. 在情绪很强时先暂停，等身体平稳后再沟通。
4. 把请求说具体，例如“希望你今天晚上回复我”比“你能不能在乎我一点”更容易执行。
5. 接受对方可能无法完全满足自己，同时也保留自己的选择权。

如果用户处在持续被控制、羞辱、威胁或暴力的关系中，应优先鼓励其联系可信任的人、学校或单位支持系统、当地专业机构或紧急服务，确保现实安全。
