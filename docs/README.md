# 心理知识库演示文档

本目录提供一组可直接上传到后台知识库的 `.md` 文档，用于演示 Hybrid RAG 的知识切片、向量化、BM25 检索和问答增强能力。

建议上传顺序：

1. `stress_management.md`
2. `anxiety_relaxation.md`
3. `sleep_and_emotion.md`
4. `cognitive_reframing.md`
5. `interpersonal_communication.md`
6. `crisis_support.md`

演示问题示例：

- 我最近压力很大，经常睡不着，可以怎么办？
- 长期焦虑时可以用哪些放松方法？
- 什么是灾难化思维，怎么调整？
- 和朋友沟通总是吵架，有什么表达方式？
- 如果一个人说自己撑不住了，应该怎么回应？

这些文档仅用于心理健康科普、陪伴式建议和系统演示，不用于医学诊断或治疗。
