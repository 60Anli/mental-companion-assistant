# 认知重评与常见思维模式

认知重评是指在情绪强烈时，尝试识别自己对事件的解释，并寻找更平衡、更有帮助的看法。它并不是强迫自己乐观，也不是否定痛苦，而是帮助人从单一、绝对化的想法中稍微退一步。

常见思维模式：

1. 灾难化  
把可能的困难直接想成最坏结果。例如“一次没做好，我以后都完了”。

2. 非黑即白  
把事情看成只有成功或失败、好或坏。例如“我没有做到最好，所以我就是没用”。

3. 读心术  
在没有充分证据时认定别人一定在否定自己。例如“他没回消息，肯定讨厌我”。

4. 过度概括  
用一次经历推断所有未来。例如“这次被拒绝，以后也没人会认可我”。

5. 应该化  
用过高标准要求自己。例如“我应该一直情绪稳定，不能脆弱”。

认知重评步骤：

1. 写下事件：客观描述发生了什么，不加入评价。
2. 写下自动想法：我当时脑子里冒出的第一句话是什么？
3. 标出情绪：焦虑、难过、羞愧、愤怒或无助分别有多强？
4. 寻找证据：支持这个想法的证据是什么？不支持它的证据是什么？
5. 换一种说法：有没有更平衡、更具体、更可行动的解释？

示例：

事件：我汇报时被老师指出问题。  
自动想法：我太差了，大家都觉得我不行。  
更平衡的想法：这次汇报确实有不足，但指出问题不等于否定我整个人。我可以先改最关键的两处，下次会更清楚。

陪伴式提问可以包括：这个想法有没有一点绝对化？有没有其他可能解释？如果朋友遇到同样的事，你会怎么安慰他？现在最小的下一步是什么？

认知重评适合用于压力、焦虑、人际敏感和自我否定场景。对于强烈危机或高风险表达，应优先进行安全支持，而不是讨论认知技巧。
